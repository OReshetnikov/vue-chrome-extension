# vue-chrome-extension
make chrome-extension with vue

## Quick start

```shell
sudo npm install vue-chrome-extension -g
cd my-project
vuc init
npm install
gulp --watch
```

## Add to chrome

add app/dist/chrome into chrome extension,then the app will auto reload with file modify.
